# RoofIQ Static v2

Clean static Vercel deployment. No Node, no npm, no build step.

## Deploy
1. Unzip this folder.
2. Upload the folder contents to the root of your GitHub repo.
3. Vercel will deploy as a static site.

## Files
- `index.html` - dashboard shell
- `styles.css` - layout and map styling
- `app.js` - filters, clicks, selected market logic
- `market-data.js` - 50-market modeled starter dataset

## Notes
This is modeled starter data for UI testing, not verified source-grade market intelligence.
